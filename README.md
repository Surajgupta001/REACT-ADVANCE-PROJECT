# React Advance Projects

This repository contains a collection of advanced React projects demonstrating modern web development techniques, animations, and 3D graphics.

## Projects

### 1. MacBook GSAP Website

**Path:** [`Mackbook Gsap Website/`](./Mackbook%20Gsap%20Website)

A high-fidelity 3D product showcase for the MacBook Pro, designed to replicate the premium feel of Apple's landing pages.

- **Tech Stack:** React 19, Three.js (R3F), GSAP, Tailwind CSS v4, Zustand.
- **Highlights:**
  - Interactive 3D model with configuration options.
  - Smooth, complex scroll-based animations (ScrollTrigger).
  - Responsive, modern UI architecture.
